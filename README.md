# Mercy Control Center

A lightweight dashboard prototype for a Telegram Mini App style control center for personal AI agents.

## Included
- Agent overview cards
- System overview panel
- Agent detail panel
- Static demo-safe data

## Privacy
This published bundle intentionally excludes:
- local tokens
- local file system paths
- personal identifiers
- session ids / routing internals
- live local OpenClaw data

## Local development
For private/local real data mode, use the original workspace version instead of this publish bundle.
